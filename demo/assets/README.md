# Assets Folder

This folder is for storing any images or assets for the Grid Auto website.

## Current Status
All icons are currently inline SVGs in the HTML for performance.

## If you need to add images:
1. Keep images small (max 200KB per image)
2. Use WebP or optimized JPEG/PNG formats
3. Consider using SVG for logos and icons
4. Add alt text for accessibility

## Recommended tools for image optimization:
- [Squoosh](https://squoosh.app/) - Free online image optimizer
- [TinyPNG](https://tinypng.com/) - PNG and JPEG compression
- [SVGOMG](https://jakearchibald.github.io/svgomg/) - SVG optimization
